Header

CSS

Beyond the styles, I used CSS to create the transitions of the links. I have used it to modify the links on the dropdowns too. The header is positionated as relative and the box that works as the background of the title is absolute positioned. Using the z-index property of the text boxes I made them visible.

In the part of the header under the nav, I have displayed a container as grid in combination with a div to create the image in the background of the title.

Other resources

I created the dropdowns of the nav using bootstrap and some extra jQuery code i took from a snippet to make the secondary dropdowns work. The height property of the header boxes that contain the elements for the title and the one that contains the background, is set with jQuery.
